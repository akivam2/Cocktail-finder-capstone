# Cocktail Finder (Capstone Project)

This is a Node.js + Express project that allows users to:
- Search for cocktails by name
- Get a random cocktail
- View full cocktail details including ingredients and instructions

## Technologies Used
- Node.js
- Express
- EJS
- Axios
- TheCocktailDB API

## How to Run the Project

1. Install dependencies:
   npm install

2. Start the server:
   nodemon index.js

   (or)
   node index.js

3. Open your browser and go to:
   http://localhost:3000
